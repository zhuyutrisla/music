# music
移动端的音乐播放器


### 采用动态rem方案：
1. 获取页面宽度大小 
2. 将10rem设置为页面宽度，并写入css
3. css所用单位均采用rem
4. 页面的图标大小、盒子布局会随着手机屏幕大小自适应；


** 如果要在电脑端打开，以chrome为例，可以打开开发者工具切换至移动端预览并刷新页面(重新获取页面宽度)  **
